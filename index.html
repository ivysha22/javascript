introduction to html

javascript is an interprated language hence we need an interpreter to excecute js code

the browser comes with built in js engine (has an interpretor ) that translates javascript to machine code feeds the processor and gets back feedback/result of the execution

we can also download other js runtimes/translators/interpretors like node js deno js and bun js to execute js outside the browser

js on the browser

to use js on the browser create a html file with a script element

the script element can contain js code in it or a src attribute with the url path to a javascript.js file
